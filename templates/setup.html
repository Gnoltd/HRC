📋 Setup Checklist

Step 1: Create Firestore Database

The Firestore database needs to be created in your Firebase Console.

  1. Visit: Firebase Console - Datastore Setup
  2. Click "Select Firestore" or "Create Database"
  3. Choose "Firestore Native Mode"
  4. Select a location (e.g., us-central1 or nearest to you)
  5. Click "Create Database"

⏱️ This usually takes 1-2 minutes to complete.

Step 2: Configure Security Rules

Set up Firestore security rules to allow read and write operations.

  1. In Firebase Console, go to Firestore Database → Rules
  2. Replace the existing rules with:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /projects/{project} {
      allow read, write: if true;  // For development only
    }
  }
}
  1. Click "Publish"

⚠️ Important: These rules allow anyone to read/write. For production, implement proper authentication.

Step 3: Download Service Account Credentials

Download the Firebase Admin SDK credentials for backend operations.

  1. Go to Project Settings → Service Accounts
  2. Click "Generate New Private Key"
  3. Save the downloaded JSON file as firebase_credentials.json
  4. Place it in the project root directory (same folder as app.py)
  5. Restart the Flask application

Step 4: Verify Setup

After completing the steps above:

  1. Restart the Flask application
  2. Refresh this page to check the connection status
  3. Try submitting a test project from the Submit Project page
